# spacex_launch_data_eda
Exploratory Data Analysis (EDA) and data visualization of the SpaceX Falcon 9 launch dataset using Python, Pandas, Matplotlib, and Seaborn to identify factors influencing first-stage landing success.

# SpaceX Launch Data Exploratory Data Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on the SpaceX Falcon 9 launch dataset to identify the factors that influence successful first-stage landings. Using Python and popular data science libraries, the analysis explores relationships between payload mass, launch sites, orbital destinations, flight numbers, and mission outcomes through statistical summaries and visualizations.

## Objectives

- Clean and prepare the launch dataset
- Explore feature distributions and relationships
- Visualize launch success across different variables
- Analyze trends over time
- Prepare data for future machine learning models

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis

- Launch Site vs Landing Success
- Payload Mass vs Landing Success
- Flight Number Analysis
- Orbit Type Analysis
- Yearly Launch Success Trends
- Feature Engineering using One-Hot Encoding
